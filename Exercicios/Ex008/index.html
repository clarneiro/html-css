Principais formatações:

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag B (não semântica)

Já nesta frase, temos um termo em destaque usando a tag STRONG (semântica)

Itálico / Ênfase

Aqui temos um termo em itálico usando a tag I (não semântica)

Já nesta frase, temos um têrmo em ênfase, usando a tag EM, semântica.

Texto marcado

Podemos criar, também, um texto marcado usando a tag MARK

Podemos também criar um texto marcado, modificando a cor, utilizando os recursos do CSS - tag style.

Outra opção é, no campo do CSS, declarar a tag diretamente no HEAD, para estabelecer uma cor, válida para todas as tags MARK da página.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo.

Texto deletado / excluído

Podemos marcar um texto como excluído para indicar que já não vige mais. Pode ser ainda lido mas não considerado!

Texto inserido

Podemos marcar um texto inserido para dar uma ênfasse e indicar que ele foi inserido posteriormente.

Existe também o sublinhado, com a tag U, que é não semântica!

Sobrescrito

Para inserir caractéres de fórmulas matémática, por exemplo: X20+3

Subscrito

Para inserir coisas como a fórmula da água: H2O
